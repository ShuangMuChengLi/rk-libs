# 正则表达式汇总
## 移除代码中的// console.log
```
(\/\/\s)?console.log\(.+\);$
```
## 正则匹配出html中所有的超链接
```
/(https|http):\/\/[^"'\\]+/gm
```
## 正则匹配出html中所有带图片的html片段
```
/src=['"]?([^'"]*)(\.jpg|.png)['"]?/gm
```
## 正则匹配出html片段中的图片
```
/src=['"]?([^'"]*)['"]?/i
```
## 匹配两个div间的内容
```
/(?<=<div>)((?!<\/div>).*?)(?=<\/div>)/
```
